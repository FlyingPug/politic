# politicQuiz
Данный проект представляет собой экспертную систему по определению политической идеологии  пользователя

Вопросы и их веса легко редактируемы, также имеется возможность создания дополнительных
списков вопросов.

примеры запроса на сайт

```JSON

    "answers" : [0,2,2,2,2,2,0,0]
}
```
ответ в таком случае:
```JSON

  "ideologyName": "centrism",
```
